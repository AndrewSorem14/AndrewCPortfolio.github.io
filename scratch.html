Scratch Content

Name Assingment

Description:

The name assignment was to use scratch to write your name and have each of the single letters have their own unique behaviors. For extra credit hace the letters go back to their original place.

Concepts Learned:

I learned how to use event blocks like green flag is clicked, when ___ key is pressed, etc. I learned conditional statements like if statements and I also learned how to use loops like repeat and forever. I also learned how to pick out sprites and make them move to specific locations.

Computer Buying Project

Description:

This was a project where we had to interview someone on what kind of things they would want if someone bought a computer for them. Then, we had to use scratch to show what computer matches what they said.

Concepts Learned:

I learned the importance of switching the background at a certain time so the characters say what they are supposed to when the backdrop switches.

Final Project

Description:

This was a project where we had to show all that we learned from scratch. For my final project I made a game where you had to score 1000 points in 60 seconds. If you win you will move on to the next round. Each round you have to shoot a UFO by pressing the space key.

Concepts learned:

I learned how to use custom made blocks for movement and to set/change variables.
